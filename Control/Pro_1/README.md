
Lab 1: Process Dynamic Characteristics

Objective: Learn to extract a process model — gain, dead time, and time constant — from step-test data.

What's Covered


Estimating process gain (K_P), dead time (θ), and time constant (τ) from a step response
Comparing a "pure" first-order lag plus dead time (FOLPDT) process against a more realistic, approximated process
Exploring other process behaviors:

Negative process gain
Integrating processes (no natural equilibrium, e.g. tank level)
Runaway (open-loop unstable) processes
Inverse response ("shrink-and-swell" effect)





Key Takeaway

These three parameters (K_P, θ, τ) form the basis for calculating PID tuning parameters used in later labs.
