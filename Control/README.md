PC-ControLAB Process Control Labs

A collection of 4 laboratory exercises exploring process dynamics and PID control using PC-ControLAB, a process control simulation environment.

Labs

1. Process Dynamic Characteristics
Learn to extract a process model (gain, dead time, time constant) from step-test data, and explore different process behaviors: first-order lag, negative gain, integrating, runaway, and inverse response processes.

2. PID Controller Characteristics
Break down the PID controller into its three modes — Proportional, Integral, and Derivative — and observe each one's individual effect on controller output, including reset windup and derivative lead behavior.

3. Tuning from Open Loop Tests
Identify process parameters from open-loop step tests, then calculate and compare PID tuning using several classical methods (Ziegler-Nichols, Lambda, Ciancone-Marlin, Myke King/P-on-PV) against manual trial-and-error tuning.

4. Modifications to the Textbook PID
Explore real-world PID enhancements used in commercial controllers: derivative/proportional on measurement (vs. on error), derivative filtering, setpoint softening, interacting vs. non-interacting algorithm forms, and bumpless MAN/AUTO transfer.

Key Concepts

Process gain · dead time · time constant · FOLPDT modeling · proportional/integral/derivative action · reset windup · derivative spike · proportional kick · Ziegler-Nichols & Lambda tuning · overshoot & ITAE · setpoint softening · bumpless transfer
